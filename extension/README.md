# GitHub Radar Browser Extension

Chrome/Edge extension that shows GitHub Radar scores on project pages.

## Install (Developer Mode)

1. Open `chrome://extensions/` (or `edge://extensions/`)
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked** → select this `extension/` folder

## Features

- **Score badge**: Visit any GitHub project page — the Radar score appears in the About sidebar
- **Quick search**: Click the toolbar icon to search open-source projects

## Build for distribution

For production, export SVG icons to PNG (16/48/128px) and zip the directory.
